### 1. 核心定义 (Core Definition)

* **产品定位**：现实主义恐怖 + 资源管理策略 + 异化叙事 AVG。
* **设计口号**：*“如果生活欺骗了你，那是因为你没看用户协议。”*
* **游戏目标**：在 **40 个回合** 内，在 **HP (生存)** 与 **Gold (阶级)** 的双重压力下，平衡 **SAN (认知)**，收集档案达成结局。
* **数值设计原则**：
* **阶级壁垒**：指数级上升，确保底层难以跨越。
* **消费主义**：高效回血但降智，诱导玩家为了生存放弃思考。
* **觉醒代价**：极高的经济惩罚（SAN值薪资衰减）。

### 2. 核心循环结构 (The Locked Loop)

此流程为游戏主循环的 **不可更改骨架**：

#### 1. Day Start (回合开始)

* **UI 渲染**：读取 SAN 值加载滤镜（蓝药丸/裂痕/红药丸）。
* **阶级判定**：系统读取当前 Gold，强制划定本日【阶级】。

#### 2. Event Trigger (事件抽取)

* **疯子红利**：高 SAN 值提高触发“高风险/黑产事件”概率。
* **【薪资衰减公式】(The Awakening Penalty)**：
* *逻辑*：你越清醒，越难以忍受异化工作。
* **SAN 0-20 (麻木)**：获得 **100%** 基础薪资。你是完美的燃料。
* **SAN 21-60 (怀疑)**：获得 **70%** 基础薪资。你开始摸鱼、迟到。
* **SAN 61-100 (觉醒)**：获得 **40%** 基础薪资。你无法工作，只能勉强维持或靠疯子红利。



#### 3. Player Choice (决策环节)

* 4 选 1 决策。**D 选项锁**：仅当 SAN 达标时可见。点击触发【知识弹窗】，必须基于真实历史。

#### 4. Settlement (结算环节)

* 结算数值。若获得信物/档案，写入【黑匣子】。

#### 5. Bill Phase (账单环节)

* **触发机制**：每日 **8%** 概率触发。若 `Gold < 0`，概率翻倍至 **16%**（屋漏偏逢连夜雨）。
* **债务模式**：`Gold < 0` 时，普通商店锁死，SAN 增长减半，高利贷启动。

#### 6. GOLD Status Check (状态检查)

* **阶级跌落**：连续 3 天 `Gold < 0` -> 强制跌落为流浪汉（资产归零）。
* *保护*：跌落瞬间，债务计数器重置为 0。


* **强制拆解**：`Homeless` 且 `Debt_Day >= 3` -> 触发【人体拆解工厂】。
* *结果*：债清，HP 上限减半。若因此死亡 -> **ED-03**。



#### 7. Shop Phase (补给环节)

* **实时刷新**：`Gold > 0` 即解锁普通商店。

#### 8. Day End (回合结束)

* Day + 1。若 Day = 40 结算结局。若 HP ≤ 0 死亡。
